# React application

## Main dependencies

-   React Router
-   Styled Components
-   Babel config
-   Prettier config

## Setup

    npm i

## Development

    npm start

Runs development server at `localhost:8080`.

## Deployment

Bump the version of your application.

    npm version patch

This command will run your tests, create a production build at the root of your project, commit and push it.
